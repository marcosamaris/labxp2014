Como configurar o ambiente?

	Acesse o arquivo tutorial.txt


Como configurar o eclipse para o code style utilizado?

	Abra Window > Preferences
	PHP > Code Style > Formatter > Active Profile: PHP Convenctions > Show
	Indentation > Tab Policy
		Mude Tab Policy de Tab para Space
		Mude Indentation Size para 4
		Clique em Ok
		Save o novo padrão
		
Como aplicar o novo code style a um arquivo antigo?
	
	No eclipse, tecle CTRL + SHIFT + F

Como criar uma página de settings no admin para um plugin?
	Crie um diretório views > default > plugins > <nome_diretorio_plugin> > settings.php
	Para alterar a função que é chamada quando o usuário clica no save, chame:
	elgg_register_plugin_hook_handler ( 'action', 'plugins/settings/save', '<nome_função_save_que_vc_criou>' );


O que é o river?
	River é a página activity (existente no elgg original).
	Não é um plugin. O atalho no menu foi apenas retirado dessa versão do elgg. 
	Ver: http://docs.elgg.org/wiki/River
	
O que são os arquivos existentes no diretório languages?
	http://learn.elgg.org/en/latest/guides/i18n.html
	São os arquivos de tradução que o php utilizada, sendo descrita como "palavra-chave" => "texto a ser exibido".
	Use elgg_echo("palavra-chave") para exibir a tradução.
	É possível adicionar vários idiomas no elgg.
